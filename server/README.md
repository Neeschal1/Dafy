# Dafy Backend
Dafy is a backend application for a second-hand product marketplace that allows users to buy and sell pre-owned products easily. It’s designed to make affordable products accessible to middle and lower-middle-class families while providing a secure, modern e-commerce experience.

## 🛠 Tech Stack
- Python
- Django & Django REST Framework (DRF) – for API development
- PostgreSQL – as the database (optional: SQLite for dev)
- Stripe – payment gateway integration for secure transactions
- LangChain – AI integration planned for Phase 2 (e.g., intelligent recommendations, chatbot support)

## ⚡ Features
### Core Features (Phase 1)
- User authentication & authorization
- CRUD operations for products
- Product categories and filtering
- Buying & selling of second-hand products
- Stripe integration for secure payments
- Order management
- User profile management

### Planned Features (Phase 2)
- AI-powered recommendations via LangChain
- Chatbot support for buyers & sellers
- Personalized product suggestions
- Enhanced search using AI embeddings
